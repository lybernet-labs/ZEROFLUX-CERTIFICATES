## Installation

1. Upload the `zeroflux-certificates/` folder to `/wp-content/plugins/`
2. Activate via **Plugins → Installed Plugins**
3. Go to **Certificates** in the WordPress admin sidebar

---

## Shortcode

Place this on any page or post:

```
[zf_verify]
```

### Examples

```
[zf_verify title="Check Your Certificate"]
```
